# 📦 JavaScript Project – Dynamic Array Simulator

This project is a Dynamic Array Simulator built using HTML and JavaScript.

It allows users to create a dynamic array and perform various JavaScript array operations interactively.

Run Project
</a>

## ⚙️ Features

✅ Dynamic Array Creation  
✅ Push Element  
✅ Pop Element  
✅ Shift Element  
✅ Unshift Element  
✅ Splice Operation  
✅ Slice Operation  
✅ Map Operation  
✅ Filter Operation  
✅ Reduce Operation  
✅ Reverse Array  
✅ Sort Ascending (A → Z)  
✅ Sort Descending (Z → A)  
✅ Show Array Length  
✅ Live Array Display  

---

## ▶️ How to Use

1. Enter the array size.
2. Click **Submit**.
3. Enter all array elements.
4. Click **Create Array**.
5. Perform any array operation using the available buttons.
